## 1. 概述与设计决策

### 1.1 任务定义

给定深度学习模型（DinkNet34_less_pool_DualHead_Freq）输出的：
- 草线概率图：`(H, W)` float32, 范围 [0, 1]
- 植被概率图：`(H, W)` float32, 范围 [0, 1]

输出草方格区域的：
1. 草线总长度（米）
2. 每个格子的破损状态（完好 / 轻度 / 重度 / 损毁 / 假性破损）
3. 真性破损 vs 植被遮挡的分类
4. 连片破损区域（面积、边界）

### 1.2 五个核心架构决策

| # | 问题 | 决策 | 原因 |
|---|------|------|------|
| 1 | 破损基准是什么 | 用规则生成 G_theoretical（不靠 ML inpainting） | ML 补全会填出假边，导致永远检测不到真破损 |
| 2 | 是否先旋转再分析 | 是，先旋转到网格主轴坐标系再做图论 | 斜坐标系下步长/走廊匹配误差大 |
| 3 | 图节点选什么 | 同时维护两套图：G_intersection（底层）+ G_cell（上层） | 项目目标是"破损格子数"，只有格子级图能直接回答 |
| 4 | 补全 vs 图论顺序 | 先图论后对比，永远不在补全后的掩码上建图 | 路线A会丢失所有真破损 |
| 5 | 连片破损如何定义 | 用 Void 多边形（缺失边围成的闭合区域）检测 | 比单纯数断边数更鲁棒，且有明确面积指标 |

### 1.3 规格参数

| 参数 | 值 | 来源 |
|------|----|------|
| GSD | 0.006388 m/px | 技术文档 v2.0 |
| 草方格标准尺寸 | 2m × 2m ≈ 313 × 313 px | 技术文档 v2.0 |
| 草线宽度（像素） | 5–10 px | 经验值 |
| 连片破损面积阈值 | > 20 m²（约 5 个格子） | 校准值（原 5 m² 偏小） |
| 植被遮挡判定阈值 | veg_coverage ≥ 0.6 | 经验值（避免植被边缘噪点误判） |
| 真性破损植被阈值 | veg_coverage ≤ 0.2 | 经验值 |
| 边覆盖率完好阈值 | coverage_ratio ≥ 0.7 | 经验值 |
| 走廊匹配半宽 | step / 4 px（约 78 px） | 经验值，平衡召回率和精度 |

## 3. 阶段 1 — 草线/植被后处理

**文件**：`postprocess.py`
**输入**：草线概率图 `(H, W)` float32, 植被概率图 `(H, W)` float32
**输出**：草线二值掩码 `(H, W)` uint8, 植被二值掩码 `(H, W)` uint8

### 3.1 草线后处理

```python
def clean_grass_mask(grass_prob, threshold=0.5,
                     close_radius=3, min_area=20):
    """
    Args:
        grass_prob: (H, W) float32, 范围 [0, 1]
        threshold: 二值化阈值，默认 0.5
        close_radius: 闭运算结构元素半径（像素），用于闭合小断裂
        min_area: 连通域面积过滤阈值，小于此值的噪点剔除
    Returns:
        mask: (H, W) uint8, 0/255
    """
    # Step 1: 阈值二值化
    binary = (grass_prob > threshold).astype(np.uint8) * 255

    # Step 2: 形态学闭运算（disk），修复 1~3px 的窄 gap
    # 注意：用 disk 而不是 line，是为了不引入方向偏好
    kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (close_radius*2+1,)*2)
    closed = cv2.morphologyEx(binary, cv2.MORPH_CLOSE, kernel)

    # Step 3: 形态学开运算（去掉孤立的噪点斑块）
    open_kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (3, 3))
    opened = cv2.morphologyEx(closed, cv2.MORPH_OPEN, open_kernel)

    # Step 4: 连通域面积过滤
    num_labels, labels, stats, centroids = cv2.connectedComponentsWithStats(
        opened, connectivity=8)
    filtered = np.zeros_like(opened)
    for i in range(1, num_labels):  # 跳过背景（label=0）
        if stats[i, cv2.CC_STAT_AREA] >= min_area:
            filtered[labels == i] = 255

    return filtered
```

**设计理由**：
- 闭运算用 disk 而非 line：草线在图像里是近似水平和垂直的，但旋转角度未知（大约 0°~20°），用 disk 可以无方向偏好地闭合各方向的 gap
- 闭运算半径 3px：对应 GSD=0.006388 m/px，约 0.019m ≈ 2cm 的断裂可被修复
- 面积过滤阈值 20px：约 0.8 mm² 的噪点，远小于最小草方格（313×313 px），安全过滤

### 3.2 植被后处理

```python
def clean_veg_mask(veg_prob, threshold=0.5, close_radius=3):
    """
    Args:
        veg_prob: (H, W) float32, 范围 [0, 1]
        threshold: 二值化阈值，默认 0.5
        close_radius: 闭运算结构元素半径，填补植被内部小孔
    Returns:
        mask: (H, W) uint8, 0/255
    """
    binary = (veg_prob > threshold).astype(np.uint8) * 255
    kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (close_radius*2+1,)*2)
    closed = cv2.morphologyEx(binary, cv2.MORPH_CLOSE, kernel)
    return closed
```

**设计理由**：植被是区域目标，不像草线那样需要细线保持，对形态学操作不敏感。闭运算仅用于填补植被内部小孔，防止边缘噪点干扰后续重叠率计算。

## 4. 阶段 2 — 骨架化与边追踪

**文件**：`skeleton.py`
**输入**：草线二值掩码 `(H, W)` uint8
**输出**：`G_actual`（NetworkX Graph），骨架 `(H, W)` uint8

### 4.1 骨架化

使用 `scikit-image` 的 `skeletonize`（基于 Lee 算法，等效于迭代 Zhang-Suen 但更鲁棒）：

```python
from skimage.morphology import skeletonize

def skeletonize_mask(mask):
    """
    Args:
        mask: (H, W) uint8, 0/255
    Returns:
        skeleton: (H, W) uint8, 1=骨架
    """
    binary = (mask > 0).astype(np.uint8)
    skeleton = skeletonize(binary, method='lee').astype(np.uint8)
    return skeleton
```

**注**：Zhang-Suen（纯 numpy 实现）适合学习原理，生产环境推荐 `skimage.morphology.skeletonize`（Lee 算法），支持 `method='lee'`，速度更快、边界处理更好。

### 4.2 毛刺去除

骨架化后残留的短毛刺（spurs）会影响节点检测，需要在节点检测前剔除：

```python
def remove_spurs(skeleton, min_spur_length=10):
    """
    动态阈值毛刺去除：从端点出发 DFS，分支长度 < min_spur_length 的剔除

    Args:
        skeleton: (H, W) uint8, 1=骨架
        min_spur_length: 毛刺长度阈值（像素），默认 step/3 ≈ 10px
    Returns:
        cleaned: (H, W) uint8
    """
    skel = skeleton.copy()
    h, w = skel.shape
    visited = np.zeros_like(skel, dtype=bool)

    # 找所有端点（度=1）
    endpoints = []
    for r, c in zip(*np.where(skel > 0)):
        degree = count_neighbors(skel, r, c)
        if degree == 1:
            endpoints.append((r, c))

    # 从每个端点 DFS，标记短分支
    for ep in endpoints:
        path = []
        dfs_spur(ep, skel, visited, path)
        if len(path) < min_spur_length:
            for r, c in path:
                skel[r, c] = 0
    return skel
```

### 4.3 节点检测

```python
def detect_nodes(skeleton):
    """
    检测骨架图中的节点

    节点类型：
        - endpoint: 度=1，骨架线在此终止
        - junction: 度>=3，交叉点或分叉点

    注：度=2 的像素是边内部点，不作为节点

    Args:
        skeleton: (H, W) uint8, 1=骨架
    Returns:
        cross_points: list of (r, c)，交叉点
        end_points: list of (r, c)，端点
        branch_points: list of (r, c)，分叉点（度>3 或 L 形）
    """
    cross_pts, end_pts, branch_pts = [], [], []

    for r, c in zip(*np.where(skeleton > 0)):
        degree = count_neighbors(skeleton, r, c)

        if degree == 1:
            end_pts.append((r, c))
        elif degree >= 3:
            # 进一步区分交叉点（两正交方向）vs 分叉点
            dirs = get_neighbor_directions(skeleton, r, c)
            has_h = any(d in dirs for d in ['E', 'W'])
            has_v = any(d in dirs for d in ['N', 'S'])
            if has_h and has_v:
                cross_pts.append((r, c))
            else:
                branch_pts.append((r, c))

    return cross_pts, end_pts, branch_pts


def count_neighbors(skel, r, c):
    """8邻域非零像素数"""
    h, w = skel.shape
    count = 0
    for dr in [-1, 0, 1]:
        for dc in [-1, 0, 1]:
            if dr == 0 and dc == 0:
                continue
            nr, nc = r + dr, c + dc
            if 0 <= nr < h and 0 <= nc < w:
                if skel[nr, nc] > 0:
                    count += 1
    return count
```

### 4.4 边追踪

将骨架上的节点用 BFS 连接成边，记录每条边的像素列表：

```python
from collections import deque
import networkx as nx

def trace_edges(skeleton, cross_pts, end_pts, branch_pts):
    """
    边追踪：沿骨架从节点走到下一个节点，记录路径像素

    Returns:
        G: nx.Graph
            节点属性: pos_px=(r,c), type='cross'|'branch'|'end'
            边属性: pixels=[(r,c),...], length_px=int, direction='H'|'V'|'diagonal'
    """
    G = nx.Graph()
    h, w = skeleton.shape

    # 建立所有节点的索引
    all_nodes = cross_pts + branch_pts + end_pts
    node_idx = {pt: i for i, pt in enumerate(all_nodes)}
    for i, (r, c) in enumerate(all_nodes):
        t = 'cross' if (r, c) in cross_pts else \
            'branch' if (r, c) in branch_pts else 'end'
        G.add_node(i, pos_px=(r, c), type=t)

    # 对每个节点出发追踪
    traced_edges = set()
    for start_idx, (r, c) in enumerate(all_nodes):
        for dr, dc in [(-1,0),(1,0),(0,-1),(0,1)]:  # 4方向出发
            nr, nc = r + dr, c + dc
            if not (0 <= nr < h and 0 <= nc < w):
                continue
            if skeleton[nr, nc] == 0:
                continue

            # BFS 追踪到下一个节点
            path = [(r, c)]
            prev_r, prev_c = r, c
            cur_r, cur_c = nr, nc
            visited_path = set([(r, c), (nr, nc)])

            while True:
                # 检查是否到达节点
                if (cur_r, cur_c) in node_idx:
                    end_idx = node_idx[(cur_r, cur_c)]
                    break

                # 找下一步（4方向，排除来路）
                found = False
                for ndr, ndc in [(-1,0),(1,0),(0,-1),(0,1)]:
                    nnr, nnc = cur_r + ndr, cur_c + ndc
                    if (nnr, nnc) in visited_path:
                        continue
                    if not (0 <= nnr < h and 0 <= nnc < w):
                        continue
                    if skeleton[nnr, nnc] == 0:
                        continue
                    path.append((cur_r, cur_c))
                    visited_path.add((nnr, nnc))
                    cur_r, cur_c = nnr, nnc
                    found = True
                    break

                if not found:
                    break

            # 记录这条边（双向）
            if (cur_r, cur_c) in node_idx:
                end_idx = node_idx[(cur_r, cur_c)]
                edge_key = tuple(sorted([start_idx, end_idx]))
                if edge_key in traced_edges:
                    continue

                path_pixels = path + [(cur_r, cur_c)]
                length_px = len(path_pixels)

                # 方向判定（用起点和终点的坐标差）
                dr_total = cur_r - r
                dc_total = cur_c - c
                if abs(dc_total) >= abs(dr_total) * 2:
                    direction = 'H'
                elif abs(dr_total) >= abs(dc_total) * 2:
                    direction = 'V'
                else:
                    direction = 'diagonal'

                G.add_edge(start_idx, end_idx,
                           pixels=path_pixels,
                           length_px=length_px,
                           direction=direction)
                traced_edges.add(edge_key)

    return G
```

**设计理由**：用 BFS 而不是 DFS 追踪边，因为 BFS 能保证找到的是最短路径（骨架上的直线段），而 DFS 可能在有噪声的骨架上绕远路。

### 4.5 完整 G_actual 构建

```python
def build_physical_graph(grass_mask, gsd=0.006388):
    """
    端到端构建 G_actual

    Returns:
        G_actual: nx.Graph
        skeleton: (H, W) uint8
    """
    skel = skeletonize_mask(grass_mask)
    skel = remove_spurs(skel)
    cross_pts, end_pts, branch_pts = detect_nodes(skel)
    G = trace_edges(skel, cross_pts, end_pts, branch_pts)

    # 添加米制属性
    for u, v, data in G.edges(data=True):
        data['length_m'] = data['length_px'] * gsd

    return G, skel
```

## 5. 阶段 3 — 网格参数估计

**文件**：`gridfit.py`

### 5.1 方向直方图双峰估计

```python
def estimate_dominant_directions(G_actual):
    """
    用方向直方图找网格的两个主方向（应相差约 90°）

    Returns:
        theta_h: 水平主方向（弧度，0° 基准）
        theta_v: 垂直主方向（弧度，0° 基准）
        H: 方向熵（用于判断是否为规则网格）
    """
    directions = []
    for u, v, data in G_actual.edges(data=True):
        u_pos = G_actual.nodes[u]['pos_px']
        v_pos = G_actual.nodes[v]['pos_px']
        dy = v_pos[0] - u_pos[0]  # row 差
        dx = v_pos[1] - u_pos[1]  # col 差
        if dx == 0 and dy == 0:
            continue
        # atan2 返回 (-π, π]，转换为 [0, π)
        angle = np.arctan2(dy, dx) % np.pi
        directions.append(angle)

    if len(directions) < 5:
        return None, None, None

    # 方向直方图（0~180°，bin=2°）
    bins = np.linspace(0, np.pi, 91)  # 90 个 bin
    hist, edges = np.histogram(directions, bins=bins)

    # 找两个最高峰
    from scipy.signal import find_peaks
    peaks, _ = find_peaks(hist, height=np.max(hist) * 0.3, distance=10)
    if len(peaks) < 2:
        return None, None, None

    # 取前两个最高峰
    peaks = sorted(peaks, key=lambda p: hist[p], reverse=True)[:2]
    peaks = sorted(peaks)  # 按角度排序

    # 两峰应相差约 90°（π/2）
    peak1_angle = edges[peaks[0]] + (edges[1] - edges[0]) / 2
    peak2_angle = edges[peaks[1]] + (edges[1] - edges[0]) / 2

    # 如果两峰接近（< 30°），说明只有单向网格，退化为单峰处理
    if abs(peak2_angle - peak1_angle) < np.pi / 6:
        return peak1_angle, peak1_angle + np.pi / 2, None

    # 计算方向熵（用于判定是否为规则网格）
    p = hist / (hist.sum() + 1e-10)
    H = -np.sum(p * np.log2(p + 1e-10))

    return peak1_angle, peak2_angle, H
```

### 5.2 旋转对齐

```python
def rotate_points(points, angle_rad, center):
    """
    将一组 (r, c) 坐标绕 center 旋转 angle_rad 弧度
    逆时针为正（与 OpenCV 一致）

    Args:
        points: list of (r, c)
        angle_rad: 旋转角度（弧度）
        center: (r_center, c_center)

    Returns:
        rotated_points: list of (r', c')
    """
    cr, cc = center
    cos_a = np.cos(angle_rad)
    sin_a = np.sin(angle_rad)

    result = []
    for r, c in points:
        # 先平移到原点
        dr = r - cr
        dc = c - cc
        # 旋转
        nr = dr * cos_a - dc * sin_a + cr
        nc = dr * sin_a + dc * cos_a + cc
        result.append((nr, nc))
    return result
```

### 5.3 步长估计

```python
def estimate_step(lengths_px, expected_px=313):
    """
    从边长直方图估计网格步长（像素）

    取主峰对应边长，排除断裂导致的短边干扰

    Args:
        lengths_px: list of 边长（像素）
        expected_px: 期望步长（像素），用于 fallback

    Returns:
        step_px: 步长（像素）
        confidence: 'high' | 'medium' | 'low'
    """
    if len(lengths_px) < 3:
        return expected_px, 'low'

    lengths_arr = np.array(lengths_px)
    # 过滤极端值（太短或太长的边不参与统计）
    q1, q99 = np.percentile(lengths_arr, [1, 99])
    filtered = lengths_arr[(lengths_arr >= q1) & (lengths_arr <= q99)]

    if len(filtered) < 3:
        return expected_px, 'low'

    hist, edges = np.histogram(filtered, bins=80)
    from scipy.signal import find_peaks
    peaks, props = find_peaks(hist, height=np.max(hist) * 0.3, distance=10)
    if len(peaks) == 0:
        return expected_px, 'low'

    main_peak = peaks[np.argmax(hist[peaks])]
    step_px = (edges[main_peak] + edges[main_peak + 1]) / 2

    # 一致性检查
    if abs(step_px - expected_px) / expected_px > 0.3:
        warnings.warn(f"step={step_px:.1f}px differs >30% from expected {expected_px}px")
        confidence = 'low'
    elif abs(step_px - expected_px) / expected_px > 0.1:
        confidence = 'medium'
    else:
        confidence = 'high'

    return step_px, confidence
```

### 5.4 相位估计

```python
def estimate_phase(rotated_nodes, step_px):
    """
    估计网格相位：在旋转坐标系下，计算所有节点坐标 mod step 的众数

    网格起点相位 = 节点坐标对 step 取模后的众数
    用于确定 G_theoretical 铺设的起始偏移

    Args:
        rotated_nodes: list of (r', c')，旋转后的节点坐标
        step_px: 步长（像素）

    Returns:
        phase_r_px, phase_c_px: 行列方向的相位偏移（像素）
    """
    rs = [p[0] for p in rotated_nodes]
    cs = [p[1] for p in rotated_nodes]

    def mode_mod(values, step):
        mods = np.array(values) % step
        # 聚类：找最密集的点
        hist, edges = np.histogram(mods, bins=np.arange(0, step + 1))
        peak = edges[np.argmax(hist)] + 0.5
        return peak % step

    phase_r = mode_mod(rs, step_px)
    phase_c = mode_mod(cs, step_px)
    return phase_r, phase_c
```

### 5.5 完整参数估计

```python
@dataclass
class GridParams:
    theta_main: float       # 旋转角（弧度）
    step_x_px: float       # 列方向步长（像素）
    step_y_px: float       # 行方向步长（像素）
    phase_x_px: float      # 列方向相位（像素）
    phase_y_px: float      # 行方向相位（像素）
    gsd_m: float           # GSD（米/像素）
    direction_entropy: float  # 方向熵（用于判定是否为规则网格）
    confidence: str        # 'high' | 'medium' | 'low'


def estimate_grid_params(G_actual, H, W, gsd=0.006388,
                         expected_step_m=2.0, expected_step_px=313):
    """
    端到端网格参数估计

    流程：
      1. 方向直方图找双峰 → theta_h, theta_v
      2. 旋转角 = theta_h（或 theta_v）
      3. 所有节点旋转 -theta → rotated_nodes
      4. rotated_nodes 的列坐标差分 → step_x_px
      5. rotated_nodes 的行坐标差分 → step_y_px
      6. 相位估计

    Returns:
        GridParams dataclass
    """
    # Step 1: 方向估计
    theta_h, theta_v, H_entropy = estimate_dominant_directions(G_actual)
    if theta_h is None:
        return None  # 无法估计，图像中没有足够的规则边

    # Step 2: 确定旋转角（取水平主方向作为旋转角）
    # 旋转后水平边应该接近 0°，垂直边接近 90°
    theta_main = theta_h  # 旋转角度

    # Step 3: 旋转节点
    center_r, center_c = H / 2, W / 2
    all_pos = [G_actual.nodes[n]['pos_px'] for n in G_actual.nodes]
    rotated_nodes = rotate_points(all_pos, -theta_main, (center_r, center_c))

    # Step 4: 步长估计（分别估计水平和垂直方向）
    h_lengths, v_lengths = [], []
    for i, (r, c) in enumerate(rotated_nodes):
        node_idx = list(G_actual.nodes())[i]
        for neighbor in G_actual.neighbors(node_idx):
            neighbor_pos = G_actual.nodes[neighbor]['pos_px']
            rn, cn = rotate_points([neighbor_pos], -theta_main, (center_r, center_c))[0]
            dr = abs(rn - r)
            dc = abs(cn - c)
            if dc > dr * 2:  # 水平边
                h_lengths.append(dc)
            elif dr > dc * 2:  # 垂直边
                v_lengths.append(dr)

    step_x_px, cx = estimate_step(h_lengths, expected_step_px)
    step_y_px, cy = estimate_step(v_lengths, expected_step_px)

    # Step 5: 相位估计
    phase_y_px, phase_x_px = estimate_phase(rotated_nodes, (step_x_px + step_y_px) / 2)

    # Step 6: 置信度
    if H_entropy is not None and H_entropy > 2.5:
        confidence = 'low'  # 方向熵高，规则性差
    else:
        confidence = 'high' if cx == 'high' and cy == 'high' else 'medium'

    return GridParams(
        theta_main=theta_main,
        step_x_px=step_x_px,
        step_y_px=step_y_px,
        phase_x_px=phase_x_px,
        phase_y_px=phase_y_px,
        gsd_m=gsd,
        direction_entropy=H_entropy or 999,
        confidence=confidence
    )
```

### 6.2 走廊匹配（核心算法）

```python
def build_pixel_set_from_skeleton(skeleton):
    """将骨架图像转为 set{(r,c)}，加速像素查询"""
    return set(zip(*np.where(skeleton > 0)))


def find_actual_edge_in_corridor(theoretical_pixels, skeleton_pixel_set,
                                  corridor_px=78):
    """
    在理论边的像素走廊内查找实际草线覆盖

    走廊 = 理论边像素的 ±corridor_px 像素范围内的所有位置

    Args:
        theoretical_pixels: 理论边上的像素坐标 list[(r,c)]
        skeleton_pixel_set: 骨架像素 set{(r,c)}
        corridor_px: 走廊半宽（像素）

    Returns:
        covered_ratio: float, [0,1]，理论边被实际草线覆盖的比例
        actual_length_px: 匹配到的实际边总长度（像素）
    """
    # Step 1: 构建走廊像素集合
    corridor = set()
    for (r, c) in theoretical_pixels:
        for dr in range(-corridor_px, corridor_px + 1):
            for dc in range(-corridor_px, corridor_px + 1):
                corridor.add((r + dr, c + dc))

    # Step 2: 求交集
    covered = corridor & skeleton_pixel_set
    covered_ratio = len(covered) / max(len(theoretical_pixels), 1)

    # Step 3: 估算实际边长度（通过 covered 像素的骨架连通分量）
    # 找 covered 区域内骨架像素的连通分量，取最大分量的长度
    if len(covered) == 0:
        return 0.0, 0.0

    # 标记 covered 区域
    covered_mask = np.zeros(skeleton_pixel_set.pop() if skeleton_pixel_set else (1, 1), dtype=bool)
    # 实际实现：构建一个 (H, W) bool 数组，只在 covered 位置标 True
    ...

    return covered_ratio, actual_length_px


def generate_theoretical_edge_pixels(r1, c1, r2, c2):
    """
    生成两个网格点之间理论边的像素列表（直线 Bresenham 算法）

    Args:
        r1, c1, r2, c2: 起止坐标（旋转坐标系下）

    Returns:
        pixels: list of (r, c)
    """
    pixels = []
    dr = abs(r2 - r1)
    dc = abs(c2 - c1)
    if dr >= dc:
        # 近似水平
        step = 1 if r2 > r1 else -1
        err = dr // 2
        for i in range(0, dr + 1):
            pixels.append((r1 + i * step, c1 + int(err / dr) if dr > 0 else 0))
            err -= dc
            if err < 0:
                c1 += 1 if c2 > c1 else -1
                err += dr
    else:
        # 近似垂直
        step = 1 if c2 > c1 else -1
        err = dc // 2
        for i in range(0, dc + 1):
            pixels.append((r1 + int(err / dc) if dc > 0 else 0, c1 + i * step))
            err -= dr
            if err < 0:
                r1 += 1 if r2 > r1 else -1
                err += dc
    return pixels
```

@dataclass
class EdgeMatch:

def build_and_match(G_actual, grid_params, bbox_px, grass_mask,
                    corridor_ratio=0.25, gsd=0.006388):
    """
    端到端构建 G_theoretical 并与 G_actual 做边级匹配

    Returns:
        G_theoretical: nx.Graph
        G_match: dict {edge_id: EdgeMatch}
    """
    # Step 1: 构建 G_theoretical
    G_theoretical = build_theoretical_graph(grid_params, bbox_px)

    # Step 2: 构建骨架像素查询集
    skeleton = skeletonize_mask(grass_mask)
    skel_pixels = build_pixel_set_from_skeleton(skeleton)

    # Step 3: 走廊半宽
    step_avg = (grid_params.step_x_px + grid_params.step_y_px) / 2
    corridor_px = max(int(step_avg * corridor_ratio), 10)

    # Step 4: 边级匹配
    G_match = {}
    for u, v, data in G_theoretical.edges(data=True):
        r1, c1 = G_theoretical.nodes[u]['pos_px']
        r2, c2 = G_theoretical.nodes[v]['pos_px']
        theo_pixels = generate_theoretical_edge_pixels(r1, c1, r2, c2)

        coverage_ratio, actual_len_px = find_actual_edge_in_corridor(
            theo_pixels, skel_pixels, corridor_px)

        theo_len_px = len(theo_pixels)
        theo_len_m  = theo_len_px * gsd
        actual_len_m = actual_len_px * gsd
        missing_len_m = theo_len_m - actual_len_m

        # 状态判定
        if coverage_ratio >= 0.7:
            status = 'intact'
        elif coverage_ratio >= 0.3:
            status = 'minor'
        elif coverage_ratio >= 0.1:
            status = 'severe'
        else:
            status = 'missing'

        G_match[(u, v)] = EdgeMatch(
            edge_id=(u, v),
            coverage_ratio=coverage_ratio,
            actual_length_m=actual_len_m,
            theoretical_length_m=theo_len_m,
            missing_length_m=missing_len_m,
            status=status,
            veg_coverage=0.0,   # 待植被分析填充
            reason='unknown'
        )

    return G_theoretical, G_match, skeleton

def compute_veg_coverage(missing_edge_pixels, veg_mask):

    计算缺失边区域的植被覆盖比例

### 8.2 Void 多边形检测

```python
def detect_void_polygons(G_theoretical, G_match, gsd=0.006388):
    """
    检测由缺失边围成的闭合 Void 多边形

    算法：
      1. 找出所有 missing/severe 边
      2. 建立缺失边的邻接图（共享端点且方向正交的两条缺失边相邻）
      3. 在邻接图上找最小闭合环（用 DFS 找环）
      4. 计算每个环的面积

    Returns:
        void_polygons: list of dict
            {'edge_ids': [...], 'area_m2': float, 'centroid_px': (r,c)}
    """
    # Step 1: 收集缺失边
    missing_edges = [e_id for e_id, m in G_match.items() if m.status in ('severe', 'missing')]

    # Step 2: 构建邻接关系
    adjacency = defaultdict(list)
    for i, (u1, v1) in enumerate(missing_edges):
        for j, (u2, v2) in enumerate(missing_edges):
            if i >= j:
                continue
            # 检查是否共享端点
            shared = {u1, v1} & {u2, v2}
            if len(shared) == 1:
                # 共享一个端点，检查方向是否正交
                d1 = G_theoretical.edges[u1, v1].get('direction', 'H')
                d2 = G_theoretical.edges[u2, v2].get('direction', 'H')
                if d1 != d2:  # H vs V
                    adjacency[(u1, v1)].append((u2, v2))
                    adjacency[(u2, v2)].append((u1, v1))

    # Step 3: DFS 找最小环（简化版：只找四边形）
    void_polygons = []
    visited = set()

    for start_edge in missing_edges:
        if start_edge in visited:
            continue
        path = [start_edge]
        visited.add(start_edge)
        found = dfs_find_quadrilateral(start_edge, adjacency, visited,
                                       G_theoretical, G_match, path)
        if found:
            void_polygons.append(found)

    # Step 4: 计算面积（用旋转后的坐标，矩形简化处理）
    for void in void_polygons:
        edges = void['edges']
        # 取四个角点
        corners = set()
        for u, v in edges:
            corners.add(G_theoretical.nodes[u]['pos_px'])
            corners.add(G_theoretical.nodes[v]['pos_px'])
        corners = list(corners)
        if len(corners) >= 4:
            # 用 bounding box 估算面积（简化）
            rs = [p[0] for p in corners]
            cs = [p[1] for p in corners]
            area_px2 = (max(rs) - min(rs)) * (max(cs) - min(cs))
            void['area_m2'] = area_px2 * (gsd ** 2)

    return void_polygons
```

### 8.3 格子级破损判定

```python
@dataclass
class CellState:
    cell_id: tuple         # (i, j)
    bbox_px: tuple         # ((r0,c0),(r1,c1))
    damage_rate: float     # [0,1]
    status: str            # 'intact'|'minor'|'severe'|'destroyed'|'pseudo_damaged'
    edge_statuses: dict    # {'top':...,'bottom':...,'left':...,'right':...}
    genuine_break_count: int
    occluded_count: int
    area_m2: float


def classify_cells(G_theoretical, G_match, grid_params,
                   void_polygons, gsd=0.006388):
    """
    将 G_theoretical 的节点派生成格子，判定每个格子的破损状态

    Returns:
        cells: list of CellState
    """
    # 从 G_theoretical 收集所有节点坐标
    # 在旋转坐标系下，节点坐标 (r', c') 的行列索引即为格子坐标
    node_by_ij = {}
    for n, data in G_theoretical.nodes(data=True):
        pos = data['pos_px']
        # 反推格子索引（用相位归一化）
        ci = int(round((pos[1] - grid_params.phase_x_px) / grid_params.step_x_px))
        ri = int(round((pos[0] - grid_params.phase_y_px) / grid_params.step_y_px))
        node_by_ij[(ri, ci)] = n

    # 收集所有格子坐标
    all_ri = sorted(set(k[0] for k in node_by_ij))
    all_ci = sorted(set(k[1] for k in node_by_ij))

    cells = []
    for ri in all_ri:
        for ci in all_ci:
            # 4个角点定义格子
            n00 = node_by_ij.get((ri, ci))
            n01 = node_by_ij.get((ri, ci + 1))
            n10 = node_by_ij.get((ri + 1, ci))
            n11 = node_by_ij.get((ri + 1, ci + 1))

            if n00 is None:
                continue

            # 该格子的4条边
            edges = {
                'top':    G_match.get((n00, n01)) if n01 else None,
                'bottom': G_match.get((n10, n11)) if n11 else None,
                'left':   G_match.get((n00, n10)) if n10 else None,
                'right':  G_match.get((n11, n01)) if n11 else None,
            }

            # 计算该格子的理论长度和实际长度
            theo_len = 0.0
            actual_len = 0.0
            genuine_break_count = 0
            occluded_count = 0
            edge_statuses = {}

            for dir_name, edge_match in edges.items():
                if edge_match is None:
                    continue
                theo_len += edge_match.theoretical_length_m
                actual_len += edge_match.actual_length_m
                edge_statuses[dir_name] = edge_match.status

                if edge_match.reason == 'genuine_break':
                    genuine_break_count += 1
                elif edge_match.reason == 'occluded':
                    occluded_count += 1

            damage_rate = 1 - actual_len / max(theo_len, 1e-6)

            # 格子级状态判定
            if genuine_break_count >= 3:
                status = 'destroyed'
            elif damage_rate >= 0.7:
                status = 'severe'
            elif damage_rate >= 0.3:
                status = 'moderate'
            elif damage_rate >= 0.1:
                status = 'minor'
            elif occluded_count >= 2 and genuine_break_count == 0:
                status = 'pseudo_damaged'  # 看起来破了，实际是植被遮挡
            else:
                status = 'intact'

            # 计算格子面积
            area_px2 = grid_params.step_x_px * grid_params.step_y_px
            area_m2 = area_px2 * (gsd ** 2)

            cells.append(CellState(
                cell_id=(ri, ci),
                bbox_px=((ri, ci), (ri+1, ci+1)),  # 简化
                damage_rate=damage_rate,
                status=status,
                edge_statuses=edge_statuses,
                genuine_break_count=genuine_break_count,
                occluded_count=occluded_count,
                area_m2=area_m2
            ))

    return cells
```

### 9.1 连通域分析（连片破损检测）

```python
def find_damage_clusters(cells, min_cluster_size=5):
    """
    在格子级图上做连通域分析，找出连片破损区域

    4-邻接：两个格子共享一条完整边（或两边都是缺失边）即为相邻

    Args:
        cells: list of CellState
        min_cluster_size: 连通域最小格子数（低于此值视为孤立破损）

    Returns:
        clusters: list of dict
            {
                'cell_ids': [...],
                'area_m2': float,
                'type': 'area_destroyed'|'degraded_patch'|'isolated_damage',
                'cascade_risk': float,  # 退化前沿比例
            }
    """
    from collections import deque

    # 建立格子坐标 → CellState 映射
    cell_map = {c.cell_id: c for c in cells}
    damaged = {c.cell_id for c in cells if c.status in ('severe', 'destroyed', 'moderate')}

    visited = set()
    clusters = []

    for cell_id in damaged:
        if cell_id in visited:
            continue
        cluster = []
        queue = deque([cell_id])
        visited.add(cell_id)

        while queue:
            cid = queue.popleft()
            cluster.append(cid)
            # 4-邻域
            neighbors = [(cid[0]+1, cid[1]), (cid[0]-1, cid[1]),
                         (cid[0], cid[1]+1), (cid[0], cid[1]-1)]
            for n in neighbors:
                if n in damaged and n not in visited:
                    visited.add(n)
                    queue.append(n)

        if len(cluster) >= min_cluster_size:
            cluster_cells = [cell_map[cid] for cid in cluster]
            area_m2 = sum(c.area_m2 for c in cluster_cells)

            # 计算退化前沿比例
            boundary_count = 0
            for cid in cluster:
                for n in [(cid[0]+1,cid[1]), (cid[0]-1,cid[1]),
                          (cid[0],cid[1]+1), (cid[0],cid[1]-1)]:
                    if n in cell_map and n not in damaged:
                        boundary_count += 1
                        break
            cascade_risk = boundary_count / len(cluster) if cluster else 0

            if area_m2 > 50:
                cluster_type = 'area_destroyed'
            elif area_m2 > 20:
                cluster_type = 'degraded_patch'
            else:
                cluster_type = 'isolated_damage'

            clusters.append({
                'cell_ids': cluster,
                'area_m2': area_m2,
                'type': cluster_type,
                'cascade_risk': cascade_risk,
                'cell_count': len(cluster)
            })

    return clusters
```

### 9.2 完整报告生成

```python
@dataclass
class AnalysisReport:
    total_actual_length_m: float
    total_theoretical_length_m: float
    global_damage_rate: float
    intact_cells: int
    minor_cells: int
    moderate_cells: int
    severe_cells: int
    destroyed_cells: int
    pseudo_damaged_cells: int
    ambiguous_cells: int
    cells: List[CellState]
    area_destroyed_regions: List[dict]
    degraded_patches: List[dict]
    isolated_damages: List[dict]
    genuine_break_total_length_m: float
    occluded_total_length_m: float
    grid_params: GridParams
    confidence: str
    # 元数据
    timestamp: str
    gsd_m: float


def generate_report(G_match, cells, clusters, grid_params,
                    confidence, gsd=0.006388):
    """端到端生成分析报告"""

    # 汇总统计
    intact_n    = sum(1 for c in cells if c.status == 'intact')
    minor_n     = sum(1 for c in cells if c.status == 'minor')
    mod_n       = sum(1 for c in cells if c.status == 'moderate')
    severe_n    = sum(1 for c in cells if c.status == 'severe')
    destroy_n   = sum(1 for c in cells if c.status == 'destroyed')
    pseudo_n    = sum(1 for c in cells if c.status == 'pseudo_damaged')
    ambiguous_n = sum(1 for c in cells if c.status == 'ambiguous')

    # 草线总长度
    total_actual   = sum(m.actual_length_m   for m in G_match.values())
    total_theo     = sum(m.theoretical_length_m for m in G_match.values())
    global_rate    = 1 - total_actual / max(total_theo, 1e-6)

    # 真/假破损长度
    genuine_len  = sum(m.missing_length_m for m in G_match.values()
                      if m.reason == 'genuine_break')
    occluded_len = sum(m.missing_length_m for m in G_match.values()
                      if m.reason == 'occluded')

    # 分类汇总
    destroyed_regions = [c for c in clusters if c['type'] == 'area_destroyed']
    degraded_patches  = [c for c in clusters if c['type'] == 'degraded_patch']
    isolated          = [c for c in clusters if c['type'] == 'isolated_damage']

    return AnalysisReport(
        total_actual_length_m=total_actual,
        total_theoretical_length_m=total_theo,
        global_damage_rate=global_rate,
        intact_cells=intact_n,
        minor_cells=minor_n,
        moderate_cells=mod_n,
        severe_cells=severe_n,
        destroyed_cells=destroy_n,
        pseudo_damaged_cells=pseudo_n,
        ambiguous_cells=ambiguous_n,
        cells=cells,
        area_destroyed_regions=destroyed_regions,
        degraded_patches=degraded_patches,
        isolated_damages=isolated,
        genuine_break_total_length_m=genuine_len,
        occluded_total_length_m=occluded_len,
        grid_params=grid_params,
        confidence=confidence,
        timestamp=datetime.now().isoformat(),
        gsd_m=gsd
    )
```

```python
class TopologyAnalyzer:
    def __init__(self, gsd=0.006388, grid_size_m=2.0, expected_step_px=313):
        self.gsd = gsd
        self.grid_size_m = grid_size_m
        self.expected_step_px = expected_step_px

    def run(self, grass_prob: np.ndarray, veg_prob: np.ndarray) -> AnalysisReport:
        # 阶段1: 后处理
        grass_mask = clean_grass_mask(grass_prob)
        veg_mask   = clean_veg_mask(veg_prob)

        # 阶段2: 骨架化
        G_actual, skeleton = build_physical_graph(grass_mask, self.gsd)

        # 阶段3: 网格参数估计
        H, W = grass_prob.shape
        params = estimate_grid_params(G_actual, H, W, self.gsd,
                                      self.grid_size_m, self.expected_step_px)

        # 阶段4: 双重图构建
        G_theo, G_match, skel = build_and_match(
            G_actual, params,
            ((0, 0), (H, W)), grass_mask,
            gsd=self.gsd)

        # 植被分析填充 G_match
        fill_veg_coverage(G_match, G_theo, veg_mask)

        # 阶段6: 破损分类
        classify_all_edges(G_match)

        # Void 多边形
        void_polygons = detect_void_polygons(G_theo, G_match, self.gsd)

        # 格子级判定
        cells = classify_cells(G_theo, G_match, params, void_polygons, self.gsd)

        # 阶段7: 连通域分析
        clusters = find_damage_clusters(cells)

        # 生成报告
        confidence = params.confidence if params else 'low'
        report = generate_report(G_match, cells, clusters, params,
                                confidence, self.gsd)
        return report
```

## 12. 性能估算

基于 1024×1024 分辨率草线掩码（对应约 187 m × 187 m ≈ 3500 m² 区域，约 875 个格子）：

| 阶段 | 操作 | CPU 时间（单核） |
|------|------|---------------|
| 1 | 阈值+形态学+连通域过滤 | 0.1–0.3 s |
| 2 | skeletonize (Lee) | 0.2–0.5 s |
| 2 | 边追踪 (BFS) | 0.1–0.2 s |
| 3 | 方向直方图+峰值检测 | < 0.05 s |
| 4 | G_theoretical 构建 | < 0.01 s |
| 4 | 走廊匹配（~3500 条边） | 0.5–2 s（主要瓶颈） |
| 5 | 植被重叠计算 | 0.2–0.5 s |
| 6 | 破损分类 | < 0.1 s |
| 7 | 连通域分析 | < 0.1 s |
| **合计** | | **~2–4 s / patch** |

**大规模优化方向**：
- 走廊匹配是主要瓶颈，可用 `sklearn.neighbors.BallTree` 加速节点空间查询
- 30000 m² 图像需要分块处理（~25 块 1024×1024），并行 `multiprocessing.Pool` 可将总时间控制在 30 s 以内

## 13. 已知限制与应对策略

| 限制 | 场景 | 应对策略 |
|------|------|---------|
| 草线完全消失超过 5×5 格子 | 大面积沙埋区，参数估计不可靠 | 当 G_actual 节点数 < 10 时，标记该区域 `confidence='low'`，用已知 grid_size 填充 |
| 网格方向剧烈变化 | 地形坡度导致的非平面投影畸变 | 分块处理，每块独立估计参数 |
| 草方格非正方形（3×3m 或 1×1m） | 不同工程规格 | 从边长直方图主峰自动检测，不硬编码 2m |
| 植被密集覆盖导致整个格子不可见 | 生态极好区域 | 该格子判为 `pseudo_damaged`，不计入破损率分子 |
| 骨架噪声导致假 junction | 分割掩码质量差 | 毛刺去除 + 动态面积阈值 + junction 后处理 |
| 旋转角估计失败 | 规则网格边太少（严重退化区域） | fallback 到 0°（不旋转）+ 告警 |
